# Render deployment notes

This project is configured for Render with `render.yaml`.

Use these important values after the Render service is live:

- Build command: `bash ./build.sh`
- Start command: `python -m gunicorn journal.asgi:application -k uvicorn.workers.UvicornWorker`
- Google OAuth callback URL: `https://YOUR-RENDER-HOST/accounts/google/login/callback/`

Required Render environment variables:

- `SECRET_KEY` is generated by `render.yaml`
- `DATABASE_URL` comes from the Render PostgreSQL database
- `RENDER=true`
- `DEBUG=False`
- `GOOGLE_OAUTH_CLIENT_ID`
- `GOOGLE_OAUTH_CLIENT_SECRET`

The build script installs dependencies, collects static files, and runs migrations.

Health checks use `/healthz/`.

If you created the Render web service manually instead of from `render.yaml`, add `SECRET_KEY` yourself in Render:

- Key: `SECRET_KEY`
- Value: generate a long random value, for example with `python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"`
